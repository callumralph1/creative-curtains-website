Accessibility Statement.

Our commitment

Creative Curtains and Blinds is committed to making this website accessible to as many people as possible. We aim to conform to WCAG 2.1 Level AA.

How accessible this website is

We believe this website is broadly accessible. Specifically:

  • All images have descriptive alt text
  • Page structure uses semantic HTML headings
  • All interactive elements are keyboard accessible
  • Colour contrast meets WCAG 2.1 AA minimum requirements
  • A skip-to-main-content link is provided at the top of every page
  • Forms have associated labels

Known issues

We are not aware of any significant accessibility issues. If you encounter a problem, please tell us.

Feedback and contact

If you encounter an accessibility barrier on this website, please contact us:

  • Email: callum@creativecurtains.uk.com
  • Telephone: 01323 844555

We will aim to respond within two working days.

Technical information

This website is built using HTML5 and CSS3. It does not use JavaScript frameworks. It is a static website hosted on Cloudflare Pages.
